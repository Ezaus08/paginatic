  • PHP (Personal Home Page): Lenguaje de programación de código abierto que se ejecuta en el servidor web, permitiendo generar contenido dinámico para sitios y aplicaciones en línea.

  • JSP (JavaServer Pages): Tecnología de Java que combina HTML o XML con código Java en el servidor para crear páginas web dinámicas sin separar excesivamente lógica y presentación.

  • ASP (Active Server Pages): Tecnología de Microsoft para generar páginas web dinámicas e interactivas en el servidor, con capacidad de conectarse a bases de datos y personalizar contenido en tiempo real.

  • Java: Plataforma y lenguaje de programación creado por Sun Microsystems en 1995, utilizada para desarrollar una gran variedad de servicios y aplicaciones confiables en el mundo digital.
